
BBN3 en Español - Versión Singleplayer

"BBN3_pve_Spanish.bps" es la versión de un solo jugador de BBN3. Incluye muchas mejoras al motor de batalla, y algunos cambios de calidad de vida unicos para la campaña. Ninguna de las características "exclusivas de PVP" que estan en la lista de cambios se encuentra presente en esta versión de BBN3.

-----------------
  Cómo Usar
-----------------

Usa el programa flips.exe incluido para parchar BBN3_pve_Spanish.bps sobre un ROM limpio de BN3 Blue.

Atento: la traducción en Español no es compatible con el Aleatorizador de BN3.

"bbn3_pve_randomize.bps" es compatible con el aleatorizador. No contiene ninguna edición de texto de BBN3, lo que desafortunadamente significa que no incluye las traducciones al Español.

El aleatorizador se puede encontrar aquí:
http://ngplus.net/index.php?/files/file/37-mega-man-battle-network-3-randomizer/
